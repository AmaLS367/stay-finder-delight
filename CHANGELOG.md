# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-01-11

### Added

- CHANGELOG.md for tracking project changes
- CONTRIBUTING.md with contribution guidelines

## [0.1.0] - 2026-01-11

### Added

- Initial release with GitHub Pages deployment setup
- Complete booking UI demo with React, TypeScript, and Vite
- HashRouter-based routing for static hosting compatibility
- LocalStorage-based data persistence (wishlist, bookings)
- Comprehensive documentation (English and Russian)
- CI/CD pipeline with GitHub Actions
- Testing setup with Vitest and Testing Library
- Code quality tools: ESLint, Prettier, TypeScript type checking
- PWA support with manifest and favicon

### Fixed

- Date input UTC drift prevention
- Trip classification using day boundaries
- ICS calendar event UTC formatting
- Race-safe localStorage updates
- Improved user feedback with toast notifications
- Print functionality reliability
